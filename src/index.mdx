# Introduction

This library provides a set of lightweight React components for the popular CSS framework [Bulma](https://bulma.io/).

The [usage](/usage) section contains full installation instructions and the [customize](/customize) section exaplins how customize
Bulma with your own CSS. 

## Declarative components

As a declarative component API react-bulma-wrapper allows you to compose your UI with the confidence provided by prop-type validation
