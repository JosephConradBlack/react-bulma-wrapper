# Hero

<PropsTable of={Hero} />

## Basic Usage

The hero component allows you to add a full width banner to your webpage, which can optionally cover the full height of the page as well.

<Playground>
  <Hero primary>
    <Hero.Body>
      <Container>
        <Title>Title</Title>
        <Subtitle>Subtitle</Subtitle>
      </Container>
    </Hero.Body>
  </Hero>
</Playground>

## Colours

The hero component supports the 7 different color props (primary, info, success, warning, danger, light and dark)

<Playground>
  <Hero info>
    <Hero.Body>
      <Container>
        <Title>Title</Title>
        <Subtitle>Subtitle</Subtitle>
      </Container>
    </Hero.Body>
  </Hero>
</Playground>
