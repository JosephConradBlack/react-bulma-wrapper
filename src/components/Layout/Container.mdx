# Container


<PropsTable of={Container} />

A simple container to center your content horizontally.

Containers support the full suite of [Typography Helpers](/bulma/typography).

## Basic Usage

<Playground>
  <Container>
    <div style={{backgroundColor: 'lightblue'}}>This container is centered on desktop</div>
  </Container>
</Playground>

## Fluid Width

<Playground>
  <Container fluid>
    <div style={{backgroundColor: 'lightblue'}}> 
      This container is <strong>fluid</strong>: it will have a 32px gap on either side, on any
  viewport size.
    </div>
  </Container>
</Playground>

## Widescreen

<Playground>
  <Container fluid>
    <div style={{backgroundColor: 'lightblue'}}>
      This container is fullwidth until the widescreen breakpoint.
    </div>
  </Container>
</Playground>

<Playground>
